# 광일푸드 홈페이지 기획안 (v1)

## 1) 목표

- 신규 B2B 고객 문의 전환
- 브랜드 신뢰도 강화
- 제품 라인업을 빠르게 이해시키는 정보 구조 구축

## 2) 핵심 타깃

- 식자재 유통사 MD
- 프랜차이즈/외식 브랜드 구매 담당자
- OEM/ODM 파트너를 찾는 식품 브랜드 운영자

## 3) 핵심 메시지

- "신선한 원재료와 표준화된 공정으로 안정적인 식품 공급을 보장한다."
- "소량 테스트부터 대량 납품까지 유연하게 대응한다."

## 4) 페이지 구조 (현재 초안 반영)

1. Hero: 브랜드 슬로건 + CTA(상담 요청, 제품 보기)
2. About: 회사 강점 3가지
3. Products: 냉동/간편식/OEM·ODM 소개
4. Process: 품질 관리 4단계
5. Contact: 문의 폼 + 연락처

## 5) 콘텐츠 우선순위

- 최우선: 회사 소개 문구, 주요 제품 카테고리, 연락 채널
- 2순위: 납품 사례, 인증 정보(HACCP, ISO 등), 주요 고객사 로고
- 3순위: 보도자료, 블로그, 채용 정보

## 6) 디자인 방향

- 톤: 신뢰감 + 신선함 + 실행력
- 컬러: 딥그린(신뢰), 웜오렌지(활력), 크림(식품/자연)
- 타이포: 굵고 명확한 헤드라인 + 읽기 쉬운 본문
- 인터랙션: 스크롤 reveal 중심의 절제된 모션

## 7) 운영/개발 체크리스트

- 연락처/주소/사업자 정보 실제 값 반영
- 문의 폼 백엔드 연동 (메일/CRM/알림)
- 개인정보 처리방침/이용약관 페이지 추가
- 성능 최적화 (이미지 WebP, lazy loading)
- SEO 기본 세팅 (title, description, OG, sitemap)

## 8) 2주 실행 로드맵 예시

1. 1~2일: 메시지/콘텐츠 확정
2. 3~5일: 디자인 확정, 컴포넌트 정리
3. 6~8일: 개발 및 폼 연동
4. 9~10일: QA, 모바일/브라우저 테스트
5. 11~14일: 배포 및 초기 트래킹 설정
